# hermes-workspace

Shared workspace for Stef + Hermes.

## Purpose
- Keep project context, docs, prompts, and scripts in one place.
- Use per-project folders under `projects/`.

## Structure
- `projects/` project-specific work
- `docs/` shared notes and decisions
- `scripts/` automation helpers
- `assets/` media and reference files

## Notes
- Do not commit secrets (`.env`, API keys, tokens).
- Keep this repository private unless intentionally sharing public content.
